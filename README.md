# Dino Platformer
> Simple 2D platform game.

The classic Chrome Dinosaur Game done with SDL.

![Dino Screenshot](screenshot.png)

### :package: Download and Compile

Requirements:

- C++ 17 or higher.
- SDL2 with Image and Mixer libraries.
- CMake v3.16 or higher
